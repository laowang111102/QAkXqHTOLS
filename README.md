# Python在线租房管理系统 python471

## 项目概述

本系统是基于Python语言的在线租房管理系统，主要服务于管理员、房东和用户三种不同的角色，实现了租房信息管理、用户交互等功能。

## 技术栈

- 后端：Python 3.8 + Django
- 前端：Vue 2
- 数据库：MySQL 5.7

## 功能模块

系统主要包括以下功能模块：

- 用户注册与登录
- 房源信息发布与管理
- 租赁订单处理
- 用户权限管理

## 系统角色

- 管理员：负责系统运维、用户管理和房源审核等。
- 房东：发布房源信息，管理租赁订单。
- 用户：浏览房源，发起租赁请求，管理个人账户。
